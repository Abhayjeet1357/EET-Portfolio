Overview

This project delivers a PLC-based control system for fuel pump automation. The cabinet integrates level sensing, permissives, interlocks, and alarm outputs to ensure safe and reliable operation in the field.

Objectives

  • Implement robust Auto/Manual modes with clear operator status.
  • Wire to neat, maintainable standards with thorough labeling.
  • Provide alarm outputs (visual + audible) for abnormal/level conditions.
  • Design a layout optimized for serviceability and future expansion.

My Role

  • Panel layout and fabrication assistance (drilling, mounting, routing).
  • PLC programming (logic for levels, alarms, and mode control).
  • Wiring & termination with attention to CSA-aware practices.
  • Documentation: photos, schematics excerpts, and report packaging.
