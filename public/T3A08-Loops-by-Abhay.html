The for in statement loops through the properties of an object: 

<p id="lhk"></p>

<script>
const person = {fname:"John", lname:"Doe", age:25}; 

let txt = "";
for (let x in person) {
  txt += person[x] + " ";
}

document.getElementById("lhk").innerHTML = txt;
</script>

The for of statement loops through the values of any iterable object:


<p id="gfh"></p>

<script>
const cars = ["BMW", "Tesla", "Mercedes"];

let text = "";
for (let x of cars) {
  text += x + "
";
}

document.getElementById("gfh").innerHTML = text;
</script>
JavaScript While Loop 


JavaScript While Loop
<p id="gg"></p>

<script>
let sk = "";
let s = 0;
while (s < 10) {
  sk += "
The number is " + s;
  s++;
}
document.getElementById("gg").innerHTML = sk;
</script>

Javascript do while loop 


Javascript do while loop

<p id="pso"></p>

<script>
let t = ""
let w = 0;

do {
  t += "
The number is " + w;
  w++;
}
while (w < 15);  

document.getElementById("pso").innerHTML = t;
</script>
